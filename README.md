# Robot Dystopia RPG 🤖

A 3rd-person action RPG set in a robot dystopia, built with Unity.

## 🎮 About This Project
This is my personal game development project for the **Stardance Challenge 2026**. I'm building a complete action RPG from scratch, focusing on:
- 3rd-person combat mechanics
- Branching narrative system
- Smooth UI/UX design
- 3D environment building

## 🛠️ Tech Stack
- **Engine:** Unity
- **Language:** C#
- **Type:** Software/Game Development
